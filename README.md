# Emotion-Aware TTS with Voice Cloning

Final Year Project

## 📌 Project Overview
This project aims to develop an emotion-aware Text-to-Speech (TTS) system capable of:

- Generating expressive speech
- Detecting / controlling emotional prosody
- Supporting voice cloning
- Reading text, stories, novels, and documents

## 🚀 Features (Planned)
- Emotion-conditioned speech synthesis
- Voice cloning from user-provided script
- Text / document input
- Expressive prosody modeling

## 🛠 Tech Stack
- Python
- Deep Learning
- TTS / Speech Models
- NLP
